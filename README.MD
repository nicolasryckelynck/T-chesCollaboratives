# GITHUB

Comment je l'ai mis en place :

- J'ai créé un dossier pour le projet
        -> Création du repository sur github
        -> git clone https://github.com/nicolasryckelynck/T-chesCollaboratives.git

- J'ai créé un dossier pour les vues (views) -> correspond à la page web, la partie visuelle
- J'ai créé un dossier pour les modèles (models) -> correspond à la table de la base de données
- J'ai créé un dossier pour les contrôleurs (controllers) -> correspond à la logique du projet
- J'ai créé un dossier pour le middleware (middleware) -> correspond au fait de savoir si l'utilisateur est connecté ou non

-> mise à jours du git
        -> git add .
        -> git commit -m "message clair et concis"
        -> git push
